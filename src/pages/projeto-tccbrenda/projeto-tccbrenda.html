Projeto TCC Brenda
Bem-vindo aos Eventos!

Este projeto tem como objetivo explorar o uso do geolocalizador em conjunto framework ionic, para dispositivos móveis. O app terá funcionalidades de encontrar situação de eventos, feiras, praias próximas do utilizador em tempo real, de acordo com cadastro e contando com avaliação de outros usuários.
